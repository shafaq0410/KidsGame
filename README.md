## 🎮 Features That Actually Slap

### 📚 English — 150 Words
- Every word has an **emoji**, a **meaning**, and an **Avengers/Cars sentence**
- Tap any word in the collection grid
- **Spell It Challenge** — type the word and get stars ⭐
- Categories: Feelings, Speed, Power, Nature, Animals, Food, Mind, Cars, Space, and more

### 🏎️ Math — Infinite Problems
- **Race car theme** — solve problems to win the race
- Choose **Addition**, **Subtraction**, or **Mixed** mode
- 4 big colorful answer buttons (no tiny text for tiny humans)
- Progress bar tracks 10 problems at a time

### ✖️ Times Tables
- Pick any number **1 to 12**
- See the **full table** displayed beautifully
- Then get **immediately quizzed** on it (pain but fun pain 😭)
